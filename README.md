# Album da Turma

Site estatico moderno e responsivo para venda de servico de criacao de album de figurinhas personalizado para turmas escolares.

## Stack

- HTML
- CSS
- JavaScript puro

## Arquivos

- `index.html`: estrutura e conteudo do site
- `styles.css`: layout, responsividade, tema claro/escuro e animacoes
- `script.js`: menu mobile, dark/light mode, FAQ, contador, pacote interativo e simulador

## Como abrir

Abra o arquivo `index.html` diretamente no navegador.

## Personalizacao

- Atualize textos, planos e depoimentos em `index.html`.
- Troque o telefone do WhatsApp pesquisando por `5500000000000`.
- Substitua os blocos da galeria por imagens reais quando tiver os mockups finais.
